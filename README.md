# 依赖 Required

* [hiredis 库](https://github.com/redis/hiredis)
* [phpredis 扩展](https://github.com/phpredis/phpredis)

* [swoole 扩展](https://github.com/swoole/swoole-src)
* [swoole_serialize 扩展](https://github.com/swoole/swoole_serialize)
